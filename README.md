# kue-job-cleaner

a simple nodejs service to remove kue job upon its completion

## Install
Install the module with:

```bash
npm install kue-job-cleaner
```

## Usage
```javascript
var kue_job_cleaner = require('kue-job-cleaner');
kue_job_cleaner.awesome(); // "awesome"
```
